  1. sqlite3相关的API
     数据库的操作
     打开数据库 sqlite3_open
     操作数据库 sqlite3_get_table / sqlite3_exec
         返回结果，需要最后释放结果 sqlite3_free_table
         不返回结果
     关闭数据库 sqlite3_close

    函数功能：打开数据库
    函数参数：
    fileName : 数据库的名字
    ppDb : 传出参数，当打开数据库成功的时候，*ppDb将会指向打开的数据库
    函数返回值：成功时，返回SQLITE_OK，失败时，返回-1；
    main函数调用它的时候，应该sqlite3 *pDb; sqlite3_open(“aa”, &pDb);
    调用该函数成功返回时，pDb这个指针指向被打开的数据库。

    函数功能：关闭数据库
    函数参数： pDb指向打开的数据库
    函数返回值： 成功时，返回SQLITE_OK，失败时，返回-1；

    函数功能：操作数据库，并且返回结果，一般多用于SQL select语句
    函数参数： pDb指向打开的数据库
    pSql : 将要执行的sql语句
    pResult : 指向返回的结果(查询到的结果，是一个二维数组)
    pRow :指向结果的行数
    pColumn : 指向结果的列数
    pErrmsg : 如果执行失败，将会指向出错消息。如果执行失败，需要调用sqlite3_free进行释放
    函数返回值： 成功时，返回SQLITE_OK，失败时，返回-1；

    函数功能：操作数据库，并且返回结果，一般多用于SQL select语句
    函数参数：
    pResult : 指向sqlite3_get_table返回的结果(查询到的结果)
    函数返回值： 成功时，返回SQLITE_OK，失败时，返回-1；

    函数功能：执行sql语句操作数据库，成功时，调用回调函数。
    函数参数：
    pDb: 指向打开的数据库
     pSql : 将要执行的sql语句
    callback : 执行sql语句成功后将会调用这个回调函数。如果结果中有多行，则每一行调用一次。
    arg :给回调函数传的第一个参数
    pErrmsg : 如果执行失败，将会指向出错消息。如果执行失败，需要调用sqlite3_free进行释放
    函数返回值： 成功时，返回SQLITE_OK，失败时，返回-1；

    函数功能：释放查询/操作数据库返回的错误信息
    函数参数：pErrmsg : 执行sql语句后的出错信息
    函数返回值： 成功时，返回SQLITE_OK，失败时，返回-1；
